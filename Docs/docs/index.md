# NADHack Introduction to Surface Mount Soldering workshop 2025

Welcome to the NADHack Introduction to Surface Mount Soldering workshop 2025!

Workshop aims:

* Show you the basic process for assembling a project using SMD components
* You try the process yourself to assemble a project
* Take home a working fully programmable, addressable LED star based on the ESP32-C3 Zero module.

# The NADHack Xmas Star V2.0

Has the following features:

* ESP32-C3 module with WiFi and Bluetooth.
* 25 individually addressable RGB LEDs.
* Programmable using Arduino IDE or can use ready made firmware such as WLED, which we will use.
* WLED includes an App to controll the star.
* Cool star shape with mounting and hanging holes.
