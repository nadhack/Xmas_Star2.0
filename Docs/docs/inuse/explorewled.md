# Explore WLED

The *WLED app* has a lot of effects and colour choices to explore. In addition, the [WLED website](https://kno.wled.ge/) has a lot of information what else is possible. 

Some notable options are:

* Homeassistant integration.
* MQTT.
* Voice control with Amazon Echo.
* Sync with Philips Hue.
* Synchronise multiple stars together.
